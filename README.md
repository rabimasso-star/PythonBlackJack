# Blackjack – Desktop App (Python)

Detta projekt är en enkel Blackjack-applikation byggd i Python med hjälp av Tkinter.
Spelet körs i ett fönster på datorn där användaren kan spela Blackjack mot datorn.

## Funktioner
- Grafiskt fönster (GUI)
- Hit och Stand-knappar
- Ny omgång utan att stänga programmet
- Datorn följer Blackjack-regeln att dra kort tills minst 17 poäng
- Ess räknas som 1 eller 11
- Fönstret öppnas centrerat på skärmen

## Regler
- Spelaren och datorn får två kort var
- Spelaren kan välja att ta fler kort (Hit) eller stanna (Stand)
- Går spelaren över 21 förlorar spelaren direkt
- När spelaren stannar spelar datorn sin tur
- Datorn drar kort tills den har minst 17 poäng
- Den som har högst poäng utan att gå över 21 vinner

## Krav
- Python 3.x
- Tkinter (ingår som standard i Python)
